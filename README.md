# Frontend

Para iniciar el proyecto AppDevFrontend, se debe realizar lo siguiente:

1- npm install, para agregar dependecias del proyecto
2- npm start para ejecutar el proyecto

Para iniciar el proyecto AppDevBackend
1- npm install
2- npm run dev
